# Laxmi Subedi — Portfolio

Personal portfolio website for Laxmi Subedi — a business management student and education counselor.

**Live site:** https://laxmisubedi.com.np

## Tech

Built with Angular 21 and Bootstrap 5.

## Run locally

```bash
npm install      # install dependencies
npm start        # dev server → http://localhost:4200
npm run build    # production build → dist/lax-portfolio-website/browser
```

## Deployment

Hosted on GitHub Pages. Every push to `main` builds and deploys automatically via GitHub Actions (`.github/workflows/deploy.yml`).
